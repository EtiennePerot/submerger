# `submerger`

## Usage:

	$ ./submerger [--exclude 'foo.*'] file1.ass file2.ass > output.ass

This will merge `file1.ass` and `file2.ass`. All subtitle styles and events full-matching regular expression `foo.*` will be excluded from the output.
